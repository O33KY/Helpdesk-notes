## Snapshots

 **No sensitive data in snapshots**

### When to use
 
- Before applying significant changes
- Before installing software
- Before Active Directory Labs
- Creating test environment
- For easy rollback after breaking 

### How to snapshot

1. Host + T (create a snapshot)  --> name it + description
3. Power off VM
4. VM manager --> Snapshots --> Restore
